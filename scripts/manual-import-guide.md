# Manual Figma Import Guide

Since the Figma API is returning "File type not supported", we'll use an alternative approach.

## Option 1: Export Frames from Figma

1. Open your Figma file: https://www.figma.com/make/BzsxHUCZe6uAbfKl6smMLv/ICP-Generator-UI
2. For each frame/page you want to import:
   - Select the frame
   - Right-click → "Copy as CSS" or "Copy as SVG" (if available)
   - Or: File → Export → Export as PNG/SVG
3. Share the exported files or design specs with me

## Option 2: Share Design Tokens Manually

Please provide:
- Color palette (hex codes)
- Typography (font families, sizes, weights)
- Spacing values
- Border radius values
- Component patterns you see

## Option 3: Frame-by-Frame Conversion

I can convert frames one at a time. Please tell me which frame to start with:
- Homepage
- Onboarding
- ICP Reveal
- Dashboard
- Collections
- Pricing
- Account
- Team Settings

Then I'll create the component based on your description or exported assets.
